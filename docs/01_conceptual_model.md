# 01 — Conceptual Model

The civilizational vector has raw and normalized forms.

- Raw vector = analytical source of truth.
- Normalized vector = visualization/comparison only.

## Core layers

```json
[
  "macro_civilization_vector_raw",
  "subcivilization_vector_raw",
  "interface_zone_vector_raw"
]
```

## State-first interpretation

At State level, some layers are estimable only approximately. The output must show data completeness, uncertainty, credible intervals, evidence trace, and comparability status.

## ADM1 interpretation

ADM1 outputs are not active in this V1. The model prepares contracts only.
