# 02 — Source-only Methodology

Question 22 = **A — public aggregate sources only**.

This V1 may use international public datasets, national official statistics, public boundary datasets, public surveys, and public administrative datasets.

It does not use questionnaires, individual responses, private social graphs, personal names/surnames, or scraped private data.
