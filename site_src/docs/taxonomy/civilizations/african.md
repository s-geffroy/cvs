# African (`african`)

!!! warning "Avertissement éthique"
    Ce profil est inféré à partir de sources publiques agrégées. Il ne doit pas être utilisé pour classer des individus réels.

**Statut** : civilisation canonique Huntington ·
**Couverture archétypes** : complète

## Synthèse

Civilisation subsaharienne hétérogène, marquée par la diversité ethnique, le syncrétisme religieux (christianisme + islam + traditions ancestrales) et l'héritage colonial.

## Description longue

Huntington note la difficulté de parler d'une 'civilisation africaine' unique. IW: TS très négatif, SE négatif. Hofstede: couverture incomplète (NGA, ZAF, KEN, GHA, ETH partiels). PDI élevé, IDV bas, IVR variable. Sous-clusters géographiques: Afrique de l'Ouest (NGA, GHA), Afrique de l'Est (KEN, ETH), Afrique australe (ZAF, ZWE, ZMB).
