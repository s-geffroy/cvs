# Orthodox (`orthodox`)

!!! warning "Avertissement éthique"
    Ce profil est inféré à partir de sources publiques agrégées. Il ne doit pas être utilisé pour classer des individus réels.

**Statut** : civilisation canonique Huntington ·
**Couverture archétypes** : complète

## Synthèse

Civilisation issue de Byzance et du christianisme orthodoxe slave, marquée par l'autocratie, le collectivisme et un héritage de modernisation tardive.

## Description longue

Distance hiérarchique très élevée (PDI souvent >85), individualisme bas-moyen (IDV 25-39), évitement d'incertitude élevé (UAI>85), restraint marqué (IVR<28). IW: TS modéré, SE négatif (survival). Inclut Russie, Ukraine, Biélorussie, Roumanie, Bulgarie, Grèce, Serbie. Frontière avec Western (Pologne, États baltes, Hongrie controversés), avec Islamic (Caucase, Balkans bosniaques).
