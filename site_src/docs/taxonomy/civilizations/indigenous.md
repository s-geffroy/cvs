# Indigenous (`indigenous`)

!!! warning "Avertissement éthique"
    Ce profil est inféré à partir de sources publiques agrégées. Il ne doit pas être utilisé pour classer des individus réels.

**Statut** : extension Huntington ·
**Couverture archétypes** : faible (low_archetype_coverage)

## Synthèse

Civilisation autochtone, non couverte par Huntington. Identité épistémique distincte (cosmologies, droit coutumier, relation à la terre), souvent en strate au sein d'États multi-civilisationnels.

## Description longue

Aucun État souverain n'est purement indigène. Présence forte en strate en BOL (aymara, quechua, guarani), ECU (kichwa), PER, GTM (maya), CAN (Premières Nations), AUS (aborigènes), NZL (Maori), NOR/SWE/FIN (Sami). Reconnaissance ONU via Permanent Forum on Indigenous Issues. Drapeau low_archetype_coverage car données IW/Hofstede absentes pour cette population transversale.
