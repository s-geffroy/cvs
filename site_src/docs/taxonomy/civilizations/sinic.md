# Sinic (`sinic`)

!!! warning "Avertissement éthique"
    Ce profil est inféré à partir de sources publiques agrégées. Il ne doit pas être utilisé pour classer des individus réels.

**Statut** : civilisation canonique Huntington ·
**Couverture archétypes** : complète

## Synthèse

Civilisation confucéenne centrée sur la Chine et ses diasporas/sphères d'influence (Singapour, Taïwan, Hong Kong, Vietnam).

## Description longue

IW: TS+ (laïcisation forte, post-confucianisme), SE négatif (survival). Hofstede: PDI élevé (58-100), IDV bas (17-26), MAS moyen (45-66), LTO très élevé (61-93). Civilisation confucéenne caractérisée par l'orientation long terme exceptionnelle (`lto`), le collectivisme, la pragmatique. Distinction interne: continentale (CHN, VNM), insulaire/diasporique (TWN, SGP, HKG).

## Cas ambigus / controverses documentées


### Corée du Sud (`KOR`)

- **Civilisations en concurrence** : sinic, buddhist, japanese
- **Discussion** : Corée du Sud: héritage confucéen (Sinic) mais identité distincte du fait de l'occupation japonaise et de l'orientation Western post-1953. Classée ici en Buddhist par défaut, mais affinités Sinic fortes.
- **Citations** : huntington_1996

### Singapour (`SGP`)

- **Civilisations en concurrence** : sinic, islamic, western
- **Discussion** : Multi-ethnique (chinois, malais, indien), mais Sinic dominant culturellement.
- **Citations** : huntington_1996
