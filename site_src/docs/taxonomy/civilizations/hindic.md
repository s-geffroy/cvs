# Hindic (`hindic`)

!!! warning "Avertissement éthique"
    Ce profil est inféré à partir de sources publiques agrégées. Il ne doit pas être utilisé pour classer des individus réels.

**Statut** : civilisation canonique Huntington ·
**Couverture archétypes** : faible (low_archetype_coverage)

## Synthèse

Civilisation hindoue centrée sur l'Inde, marquée par le système des castes, l'héritage sanskrit, et un syncrétisme religieux (hindouisme, sikhisme, jaïnisme).

## Description longue

IW: TS modérément négatif, SE négatif. Hofstede: PDI élevé (77), IDV bas-moyen (48), MAS moyen, UAI relativement bas (40), LTO modéré (51), IVR bas (26). Inclut principalement l'Inde et des populations diasporiques. Frontière avec Islamic (Pakistan, Bangladesh), Sinic (Tibet, Népal), Buddhist (Sri Lanka, Népal).
