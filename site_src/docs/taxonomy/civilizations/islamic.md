# Islamic (`islamic`)

!!! warning "Avertissement éthique"
    Ce profil est inféré à partir de sources publiques agrégées. Il ne doit pas être utilisé pour classer des individus réels.

**Statut** : civilisation canonique Huntington ·
**Couverture archétypes** : complète

## Synthèse

Civilisation organisée autour de l'Islam comme cadre normatif et culturel, hétérogène (sunnite, chiite, soufi) et géographiquement vaste (MENA, Asie du Sud, Asie du Sud-Est).

## Description longue

IW: TS négatif (traditionnel), SE négatif (survival) — quadrant Sud-Ouest. Hofstede: PDI élevé (55-100), IDV bas (14-46), UAI moyen-élevé (36-85), restraint marqué (IVR souvent <50). Distingue sous-clusters: arabe (Maghreb + Mashreq), turco-iranien (Turquie, Iran), sud-asiatique (Pakistan, Bangladesh), sud-est asiatique (Indonésie, Malaisie). Frontières: Orthodox (Caucase), Hindic (Inde-Pakistan), Sinic (Xinjiang).

## Cas ambigus / controverses documentées


### Turquie (`TUR`)

- **Civilisations en concurrence** : islamic, western
- **Discussion** : Candidat UE depuis 1999, alliance OTAN, mais réislamisation politique. Huntington classait Turquie en État 'déchiré' (torn state).
- **Citations** : huntington_1996

### Israël (`ISR`)

- **Civilisations en concurrence** : western, islamic
- **Discussion** : État juif au Moyen-Orient — ni Western ni Islamic stricto sensu.
- **Citations** : huntington_1996
