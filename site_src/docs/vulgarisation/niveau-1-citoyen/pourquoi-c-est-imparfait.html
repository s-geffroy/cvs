Pourquoi c'est imparfait

cvs est imparfait. Personne ne le cache. Cette page liste les 6 limites principales en langage courant.

1. Les données vieillissent

Les enquêtes WVS sont faites par vagues, sur plusieurs années. La vague utilisée (la 7ᵉ) date de 2017-2022. Le monde a changé depuis : pandémie, guerre, vagues populistes. Un pays qui a pris un virage net depuis 2022 n'est pas « capté » par cvs.

2. Les enquêtes ne couvrent pas tous les pays

Sur ~190 États souverains, cvs en couvre environ 62. Les autres n'ont pas de données WVS fiables. Ils sont absents de la carte (pas « zéro », pas « inconnu » — simplement non couverts).

3. Un État ≠ un peuple

cvs parle d'États souverains. Un État c'est une frontière administrative, une capitale, une représentation à l'ONU. Ce n'est pas un peuple, une nation, une diaspora, une langue. La Suisse a 4 langues, l'Inde en a des dizaines, le Canada a deux grandes communautés linguistiques — cvs les agrège.

4. Le mot « civilisation » est dangereux

Comme dit la page Qu'est-ce qu'une civilisation, ce mot a servi à justifier le pire. cvs l'emploie dans son sens statistique (un nuage de points), mais le lecteur peut être tenté de l'interpréter dans son sens politique. C'est un risque éditorial connu, expliqué en détail dans la page d'éthique officielle.

5. Les sources ont leurs biais

  • WVS : les enquêtes sont parfois faites dans des conditions politiques difficiles. Le déclaratif peut sous-estimer certains comportements (religion, vote, opinions sensibles).
  • Hofstede : les six dimensions ont été calculées à partir d'employés d'IBM dans les années 70. Elles ont été affinées depuis, mais le biais d'origine reste.
  • Inglehart-Welzel : la carte 2D est une simplification. Elle est utile, mais elle écrase beaucoup d'information.

6. Une moyenne efface les minorités

Si la population d'un pays se sépare en deux blocs très différents, la moyenne tombe au milieu — un endroit où personne n'est. cvs essaye de gérer ça en publiant aussi un « second moment » (le M(s) du site principal) qui mesure la dispersion interne, mais la moyenne reste la statistique principale.

En résumé : ce que cvs n'est pas

❌ cvs n'est PAS… ✅ cvs EST…
une vérité sur les peuples une lecture statistique d'enquêtes publiques
une note d'évaluation une carte de proximités
un classement individuel un agrégat par État
une preuve du « choc des civilisations » une base de discussion
une donnée stable dans le temps un instantané de la dernière vague WVS

En une ligne

Tout outil de mesure est imparfait. cvs assume ses limites — et invite à les relire avant de citer le moindre chiffre.
